# How to set up a server
we are using Digital Ocean as an example here. however, please feel free to use any provider of your choice.

### Click on Create at the top
![Create Droplet](https://github.com/dataanalytics3crypto/masternode_setup/blob/main/digitalocean1.png)

### Select Droplets
![Create Droplet](https://github.com/dataanalytics3crypto/masternode_setup/blob/main/digitalocean2.png)

### Scroll Down and select Ubuntu 18.04 Image
![Create Droplet](https://github.com/dataanalytics3crypto/masternode_setup/blob/main/digitalocean3.png)

### Scroll down to Choose a plan, make sure Basic is selected, select Regulat Intel with SSD not Premium.
### select 2 GB/1 CPU 50 GB 2TB Transfer, it should be around $10/month
![Create Droplet](https://github.com/dataanalytics3crypto/masternode_setup/blob/main/digitalocean4.png)

### Leave rest as default and setup a password, create a strong password and make sure to note it down in a safe place
![Create Droplet](https://github.com/dataanalytics3crypto/masternode_setup/blob/main/digitalocean5.png)

### Click Create Droplet
![Create Droplet](https://github.com/dataanalytics3crypto/masternode_setup/blob/main/digitalocean6.png)

### Get the IP address of the newly created server. it would look something like this
![Create Droplet](https://github.com/dataanalytics3crypto/masternode_setup/blob/main/digitalocean7.png)
